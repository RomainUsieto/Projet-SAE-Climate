Le Projet Climat est un projet réalisé dans le cadre d’une SAÉ (Situation d’Apprentissage et d’Évaluation) en première année – premier semestre d'ISA Num.

Il s’inscrit dans le cadre des Objectifs de Développement Durable (ODD), en lien avec l’ODD correspondant à la lutte contre le changement climatique.

Ce site web vise à sensibiliser aux enjeux climatiques en présentant de manière claire et accessible des données sur l’évolution des températures, à l’aide de visualisations réalisées en HTML, CSS et JavaScript.
